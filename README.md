## Requirements
- Android 6 and up
- OpenGLES 2.0 and up
- Magisk installed
- Moto Core Magisk Module installed https://github.com/reiryuki/Moto-Core-Magisk-Module

## Tested on
- Android 10 CrDroid ROM
- Android 11 DotOS ROM
- Android 12 AncientOS ROM
- Android 13 AOSP ROM
- Android 13 CrDroid ROM
